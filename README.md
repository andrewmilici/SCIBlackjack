# SCI Blackjack

A Blackjack game built using **SCI0**, the original Sierra Creative Interpreter engine used in classic Sierra On-Line adventure games (e.g. *King's Quest IV*).

## About

This project recreates the classic casino card game Blackjack (21) using the SCI0 engine and scripting language, built with **SCI Companion**.

## Requirements

To build or run this project, you'll need:

- [SCI Companion](https://github.com/scummvm/scicompanion) — the IDE used to develop this game
- An SCI0-compatible interpreter for testing/playing, such as:
  - [ScummVM](https://www.scummvm.org/) (supports SCI0 games)
  - Sierra's original DOS interpreter, run via [DOSBox](https://www.dosbox.com/)
